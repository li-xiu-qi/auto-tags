# 快速开始使用

## 简介

Obsidian AI Tagger 是一个 Obsidian 插件，利用大语言模型（LLM）为你的笔记自动生成标签，帮助你更好地组织和管理笔记。

## 安装

1. 下载插件文件。
2. 将插件文件夹复制到你的 Obsidian vault 的 `.obsidian/plugins/` 目录下。
3. 在 Obsidian 中启用插件。

## 配置

1. 打开 Obsidian 设置。
2. 找到 "AI Tagger" 设置页面。
3. 输入你的 OpenAI API 密钥。
4. 选择你想要使用的模型（例如 GPT-4o mini）。
5. 可选：配置自定义 Base URL（用于代理或本地模型）。
6. 可选：启用小写模式，将所有标签转换为小写。

## 使用

### 生成标签

- **工具栏按钮**：点击左侧工具栏的魔杖图标，选择文档生成标签。
- **命令面板**：使用命令 "Generate tags" 为当前文档生成标签。
- **右键菜单**：在文件或文件夹上右键，选择 "Generate tags"。

### 示例

假设你有一篇笔记内容如下：

```markdown
# 机器学习简介

机器学习是人工智能的一个分支...
```

插件会分析内容并生成相关标签，如 `#机器学习`、`#人工智能` 等，并添加到笔记的 frontmatter 中。

## 注意事项

- 确保你的 API 密钥有效且有足够的使用额度。
- 插件会根据文档内容生成标签，避免重复标签。
- 如果文档过长，插件会检查 token 限制。
